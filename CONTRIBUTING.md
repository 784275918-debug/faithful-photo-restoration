# Contributing

Contributions are welcome if they preserve the project's core principle:

> Better is not enough. The restored photo must still be faithful.

## Good Contributions

- Better quality gates.
- Better prompt templates.
- Public-domain before/after examples.
- Scripts that prevent packaging mistakes.
- Conservative local preprocessing tools.
- Documentation of failure cases.

## Avoid

- Claims of one-click perfect restoration.
- Private family photos.
- Prompts that encourage beautification or face replacement.
- Large model files committed to the repo.
- Features that make the output look impressive but less faithful.

## Example Images

Only use images that are safe to publish:

- public-domain archive photos,
- synthetic mockups,
- explicit-permission examples,
- redacted crops.

Do not submit private family photos.
